LLM Lab
=======

This repository is a playground for experimenting with large language models, coding agents, and cross-language ideas. It hosts Ruby, Python, and other tool-specific projects alongside the configs and helpers needed to iterate quickly.

Structure
---------

* `ruby/` and `python/` contain the language-specific workspaces, starting with `ruby/lib` and `python/src`.
* Hidden `.claude/` and `.codex/` directories provide places for agent configuration and artifacts.
* Add data, containers, or other experiments at will; keep the README up to date with anything that becomes stable.
